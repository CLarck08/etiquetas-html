Tipos de etiquetas


La estructura de las etiquetas en HTML

HTML está compuesto de una serie de etiquetas y otros elementos que definen todo el contenido de una web, desde las imágenes, párrafos, títulos, listas videos, definir el autor del sitio, o la descripción del mismo. Que se usan como encabezados o titulos de una pagina web, las etiquetas que se usan como estructura de la pagina son estas:

head /head, title /title, body /body, !DOCTYPE html, html /html, div /div. Tambien hay otras etiquetas para texto como palabras en negrita, cursiva, citas, entre otras; , style /style esta etiqueta se usa para cambiar la aparencia de ciertos elementos usando estilos de CSS

Etiquetas para introducir contenido

Las etiquetas para añadir recursos externos de la pagina web son los siguientes:

img Etiqueta para colocar una imagen en la página web, iframe /iframe etiqueta que sirve para vincular otro documento HTML dentro del documento principal, embed Usada para integrar una aplicación o contenido interactivo externo que no suele ser HTML. object /object Se usa para llamar a un recurso externo de la web. Este recurso será tratado como una imagen, un video o un recurso externo que es manejado por un plugin. video /video Se usa para reproducir videos en la pagina web junto a sus archivos de audio. audio /audio carga en la web un archivo de audio, source Alternativa para las etiquetas de video o audio

Etiquetas para texto

Las etiquetas para el texto son las más básicas y las más importantes para crear una pagina web:

h1 a h6 se usan para crear titulos o encabezados, p /p para parrafos de la pagina, a /a Etiqueta usada para generar un enlace en la pagina, strong /strong Etiqueta para usar negrita en una palabra o conjunto de ellas, cite /cite Para indicar una cita, autor o el titulo de una obra, sub /sub utilizadas para representar un subíndice, mark /mark Usada para resaltar o remarcar el texto, em /em usada para escribir en cursiva, hr> crea una linea divisoria en la pagina web, br Etiqueta utilizada para crear un salto de línea

Etiquetas para la creación de tablas

Las tablas son elementos dentro del diseño web que permiten organizar datos como texto, imágenes y enlaces en filas y columnas, algunas de estas etiquetas son:

table /table Etiquetas de apertura y cierre de una tabla. El resto de etiquetas de la tabla tienen que permanecer entre estas dos etiquetas, caption /caption Usada para indicar el título de la tabla, colgroup /colgroup Etiqueta utilizada para agrupar dos o más columnas de una tabla, tbody /tbody Usada para describir datos de una tabla, thead /thead Indica el bloque de filas que describen las etiquetas de las columnas, tfoot /tfoot Indica los bloques de filas que describen los resúmenes, o datos totales de una columna de una tabla. tr /tr Se usa para indicar una fila de celdas de una tabla, td /td Usada para definir una celda de una tabla, th /th Etiqueta que se usa para definir el encabezado de una celda.
